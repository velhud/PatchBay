# WorkPacket Decomposition Brief

Create WorkPackets for:

- WP-00 resolved contract and machine-readable registry;
- WP-01 identity and logical workspace foundations;
- WP-02 SQLite state, migration, and rollback;
- WP-03 Edge handshake, journal, outbox, scheduler, and claim fence;
- WP-04 operation broker, attempts, payloads, waits, and recovery;
- WP-05 worker projection and immutable fleet refs;
- WP-06 read-only worker/workspace surfaces;
- WP-07 inbox/start/batch/message mutations;
- WP-08 integration, stop, cleanup, and mutation reconciliation;
- WP-09 group close, dispositions, and successor reassignment;
- WP-10 Pro Requests;
- WP-11 complete manifest, failure/live acceptance, docs, and rollout.

Each packet must list allowed files, forbidden files, schema contracts,
migration impact, required tests, rollback point, and evidence artifact.
